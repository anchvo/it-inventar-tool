# 💻 IT Inventar Tool (Streamlit + SQLite)

Ein einfaches, aber vollständiges IT-Inventar-System zur Verwaltung von Geräten, Benutzern und Statusinformationen.

Das Projekt wurde als Lern- und Portfolio-Projekt entwickelt, um grundlegende Full-Stack-Fähigkeiten mit Python zu demonstrieren. Veränderungen und Verbesserungen werden weiterhin stetig vorgenommen.

Live Link: https://it-inventar-tool.streamlit.app/

---

## 🚀 Features

- ➕ Geräte hinzufügen (Create)
- 📋 Geräte anzeigen (Read)
- ✏️ Geräte bearbeiten (Update)
- 🗑️ Geräte löschen (Delete)
- 🔍 Suchfunktion (Filter nach Geräten, Standort, Benutzer oder Status)
- 💾 Persistente Speicherung mit SQLite
- 🎨 Einfache Web-Oberfläche mit Streamlit

---

## 🧠 Verwendete Technologien

- Python 3.12
- Streamlit (Frontend / UI)
- SQLite (Datenbank)
- UUID (stabile IDs für Datensätze)

---

## 📂 Projektstruktur

- app.py → Streamlit UI
- db.py → Datenbanklogik (CRUD Funktionen)
- inventar.db → SQLite Datenbank

---

## 🎯 Ziel des Projekts

Dieses Projekt wurde entwickelt, um:

- CRUD-Operationen praktisch umzusetzen
- eine einfache Datenbankanbindung zu lernen
- UI und Backend sauber zu trennen
- ein vollständiges Mini-Softwareprojekt zu bauen
